# CATi | 认知架构类型指标测试

基于认知神经科学与计算心理学逻辑的心智性能压测系统。不关注"性格倾向"，而是探测大脑的底层硬件配置、算力分配模式以及系统防火墙强度。

通过四个核心维度（处理模式、表征方式、逻辑载体、防御机制）交叉锁定，揭示 16 种独特的"脑内职人"架构。

## 部署

部署到 GitHub Pages：

```bash
# 1. 在 GitHub 创建仓库 CATi
# 2. 推送代码
git remote add origin git@github.com:yangcr-abaaba/CATi.git
git branch -M main
git push -u origin main
# 3. 在仓库 Settings → Pages → 选择 main 分支 → 保存
```

## 本地运行

直接用浏览器打开 `CATi.html` 即可。
